# AI_for_Silent_Speech_Recognition
AI for Silent Speech Recognition is a deep learning-based project that converts silent lip movements into text without relying on audio input. The system uses computer vision techniques to extract mouth regions from video frames and learns visual speech patterns using deep learning models. The project is trained and evaluated using the **GRID Corpus**, a widely used audio-visual speech dataset containing recordings of multiple speakers uttering structured sentences. By leveraging lip movement information, the system aims to assist individuals with speech impairments and enable communication in situations where audio is unavailable or unreliable. This project demonstrates the application of artificial intelligence, computer vision, and sequence modeling for visual speech recognition.
