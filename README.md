# AI_for_Silent_Speech_Recognition
This project focuses on developing an AI-based Silent Speech Recognition system that converts lip movements into text without using audio input. Built using computer vision and deep learning techniques, the system processes video data from the GRID Dataset to learn and recognize visual speech patterns. The goal of this project is to explore assistive communication technologies for individuals with speech impairments and to enable communication in environments where audio is unavailable or unreliable. Through this project, I gained hands-on experience in video preprocessing, lip region extraction using computer vision, dataset handling, deep learning model development, and the challenges involved in visual speech recognition systems.
